# Redmine Users Assignment Plugin

This is a plugin for Redmine to assign users by role in Issue(Author, etc) in fields of type user.

## Requirements
* Redmine 3.2.0

## Install

1.  In *plugins* folder

        git clone git@gitlab.qazz.pw:mich.malov/redmine_users_assignment.git

2.  Restart Redmine
