# Redmine Author Role Filter plugin

This is plugin which add author role filter to issues in Redmine.

## Requirements
* Redmine 3.2.0

## Install

1.  In *plugins* folder

        git clone git@gitlab.qazz.pw:mich.malov/redmine_author_role_filter.git

2.  Restart Redmine
